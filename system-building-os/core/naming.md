# Canonical Naming

The canonical name of this operating system is **Autonomous System Building OS**.

Use this name consistently throughout the repository. The following are accepted
documentation synonyms but must not create a second canonical identity:

- Autonomous Software Factory
- Planning OS
- System Building OS

The machine slug is `autonomous-system-building-os`.
