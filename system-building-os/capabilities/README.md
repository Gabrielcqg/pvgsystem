# Capabilities

The OS provides these reusable capabilities. Each maps to planning agents/skills,
implementation agents/skills, and plan templates.

- [Database capability](database.md) — Plan and implement relational/document/key-value/graph/vector storage.
- [Authentication & authorization capability](authentication.md) — Plan and implement identity, sessions, roles, permissions, org isolation.
- [Frontend & UI/UX capability](frontend-uiux.md) — Plan and implement screens, components, states, accessibility, motion.
- [Backend & API capability](backend-api.md) — Plan and implement domain services, APIs, jobs, queues, events.
- [AI workflow capability](ai-workflow.md) — Plan and implement AI behavior with strict I/O contracts and evaluation.
- [Integration capability](integration.md) — Plan and implement external integrations with fallbacks and mocks.
- [Infrastructure & environment capability](infrastructure.md) — Plan and implement environments, IaC, CI/CD, deploy, rollback.
- [Observability capability](observability.md) — Plan and implement structured logs, metrics, traces, dashboards, alerts.
- [Security capability](security.md) — Plan and implement threat model, secure config, abuse controls.
- [Performance capability](performance.md) — Plan and implement measurable performance budgets and regression detection.
- [Failure recovery capability](failure-recovery.md) — Diagnose, classify, and repair failures without infinite loops.
