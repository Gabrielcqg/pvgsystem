# How Runtime Adapters Work
Translation is **semantic, not literal**. A canonical rule like "never expose
secrets" becomes, for Claude, a CLAUDE.md rule + permission config + PreToolUse
hook; for Codex, an AGENTS.md rule + sandbox/config policy + `scan_secrets.py`
task gate (Codex has no native hooks). Adapters are regenerated by the
`generate_*_adapter.py` scripts and validated by `validate_agent_contracts.py`,
`validate_skill_contracts.py`, and `validate_runtime_config.py`.
